The Wegmans' Assistant

The technologies used were Wegmans' API, Google Cloud Platform, Google Assistant, nodeJS, JavaScript (axios, geolib libraries).

Software Application Mining 2018

Anunciar

Created a progressive web app called, 'Anunciar', a message delivery platform, restricting communication of messages in a unidirectional manner. Created it to address the issue of spam messages on group chat applications. Implemented it using Ruby on Rails.
